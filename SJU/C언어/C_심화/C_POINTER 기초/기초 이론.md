# C_Programming Pointer 기초 이론

### Pointer란?
- 포인터는 하나의 주소 값
- int *ptr; // 포인터 선언
- *ptr = arr; // 포인터의 주소값은 배열 arr[0]에 위치
- *ptr = arr + 2; // 포인터의 주소값은 배열 arr[2]에 위치

### for문을 포인터로 사용
```c++
for(int *ptr = arr; ptr < arr+10; prt++){
    printf("%d ",*ptr);
} // 배열 arr[0] ~ arr[9] 까지 출력
```
